# ⚓ APIHarbor - The API Key Health Platform for Businesses

## 🌟 Introduction
APIHarbor is the API key health platform that helps you track every API key across providers — OpenAI, Anthropic, Stripe, GitHub, AWS, Twilio, and any HTTP API — in one dashboard. Monitor key health and validity in real time, track usage and spend, and get alerts before keys expire or fail, all with encrypted storage and team collaboration built in.

## 🚀 Features

- **API Key Tracking:** Manage every key across all your providers in one place.
- **Health Monitoring:** Check key validity and health status in real time.
- **Usage Analytics:** Track usage, quota, and spend across providers.
- **Expiry Alerts:** Get notified before keys expire or fail, and rotate on time.

## 🛠️ Installation
To run APIHarbor locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-org/apiharbor.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Set up environment variables in a `.env` file:
    ```
   # app
    NEXT_PUBLIC_APP_DOMAIN=
    NEXT_PUBLIC_APP_NAME=

    # database
    DATABASE_URL=
    ```

4. Run the development server:
    ```bash
    npm run dev
    ```
